# GEOMG

An experimental UI for administering BTAA Geoblacklight JSON documents.

## Installation / Requirements

* Ruby 2.5+
* Ruby on Rails 6+
* PostgreSQL 12+
* bundle

## Development

Run this rake task

```bash
bundle exec rake geomg:server
```
